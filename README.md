# Drupal Blog Project

## Overview
This project implements a complete blog system in Drupal 10 with custom content type, modern styling, and social media integration. The solution includes a local development environment setup using Docksal.

## What Was Implemented

### Core Features
- **Custom "Blog" content type** with fields:
  - Title
  - Body/Content
  - Author
  - Publication Date
  - Featured Image

- **4 sample blog posts** covering different topics:
  - Personal introduction and background
  - Sports and hobbies
  - Technical experience with Drupal
  - Technology impact on society

- **Social media integration** with share buttons displayed on the left side of blog pages

- **Custom CSS styling** for both blog listing and individual post pages

### Technical Implementation
- Drupal 10 CMS
- Docksal local development environment
- Responsive CSS design with animations
- Git version control with clear commit history
